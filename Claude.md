# Census Studio

ArcGIS Pro toolbox for working with American Community Survey (ACS) data.

## Project Structure
- Main toolbox and Python tools in this Scripts folder
- Help documentation in separate repo

## Key Features
- Hot spot analysis with MOE-aware significance adjustment
- Interactive variable lookup
- ACS data processing and analysis tools

## Code Style
- Python for ArcGIS Pro (arcpy)
- Docstrings for all functions
- Snake_case for function and variable names

## Testing
- Test with various ACS datasets
- Verify MOE calculations and significance adjustments
